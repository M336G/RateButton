# Rate Button
**<cr>ONLY useful on GDPSs with permissions to rate levels.</c>**

**<co>This mod may not work as well on a different core than [MegaSa1nt's fork](https://github.com/MegaSa1nt/GMDprivateServer/tree/master).</c>**

----------

This mod lets GDPS owners/moderators <cb>rate levels more precisely</c> through a <cb>custom interface</c> instead of using the <co>`!rate` command</c> or the <co>moderator send popup</c>.

## Features
- <cb>Rate levels</c> & set their <cb>difficulty automatically</c>
- Set levels as <cb>rated only</c>, <cb>featured</c>, <cb>epic</c>, <cb>legendary</c>, or <cb>mythic</c>
- Toggle <cb>coins being verified or not</c>

*This mod is deeply inspired from [how RobTop rates levels](https://www.youtube.com/watch?v=_wmWuymEZDs).*
